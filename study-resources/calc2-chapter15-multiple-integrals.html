第15章 多重积分 (Multiple Integrals)

本章将一元积分的概念推广到多元函数，涵盖二重积分、三重积分、极坐标/柱坐标/球坐标下的积分、以及变量替换（Jacobian）。

15.1 矩形区域上的二重积分与累次积分

(Double and Iterated Integrals over Rectangles)

15.1.1 作为 Riemann 和极限的定义

对于定义在矩形区域

$$R = [a, b] \times [c, d] = {(x,y) \mid a \leq x \leq b,; c \leq y \leq d}$$

上的函数 $f(x,y)$，将 $R$ 划分为小的子矩形。设 $\Delta x = \frac{b-a}{m}$，$\Delta y = \frac{d-c}{n}$，则子矩形的面积为：

$$\Delta A = \Delta x \cdot \Delta y$$

在每个子矩形中选取样本点 $(x_{ij}^*, y_{ij}^*)$，则二重积分的 Riemann 和定义为：

$$\sum_{i=1}^{m} \sum_{j=1}^{n} f(x_{ij}^*, y_{ij}^*) ;\Delta A$$

当 $m,n \to \infty$（即 $\Delta A \to 0$）时，若该和的极限存在，则称该极限为 $f$ 在 $R$ 上的二重积分：

$$\iint_R f(x,y),dA = \lim_{m,n \to \infty} \sum_{i=1}^{m} \sum_{j=1}^{n} f(x_{ij}^*, y_{ij}^*) ;\Delta A$$

重要解释：$dA$ 代表面积微元（area element）。若 $f(x,y) \geq 0$，则二重积分表示曲面 $z=f(x,y)$ 下方、矩形区域 $R$ 上方的立体体积。

15.1.2 Fubini 定理 (Fubini’s Theorem)

Fubini 定理是计算二重积分的核心定理，它允许我们将二重积分转化为累次积分（iterated integral），即先对一个变量积分，再对另一个变量积分。

若 $f$ 在矩形 $R = [a,b] \times [c,d]$ 上连续，则：

$$\iint_R f(x,y),dA = \int_a^b \int_c^d f(x,y),dy,dx = \int_c^d \int_a^b f(x,y),dx,dy$$

关键点：积分次序可以交换！

累次积分的两种写法：

  • **先 $y$ 后 $x$**：
    $$\iint_R f(x,y),dA = \int_a^b \left[ \int_c^d f(x,y),dy \right] dx$$

  • **先 $x$ 后 $y$**：
    $$\iint_R f(x,y),dA = \int_c^d \left[ \int_a^b f(x,y),dx \right] dy$$

技巧：计算内层积分时，将另一个变量视为常数。

15.1.3 二重积分的性质

设 $f, g$ 在区域 $R$ 上可积：

性质公式
线性性 (Linearity)$\iint_R [cf(x,y) \pm g(x,y)],dA = c\iint_R f,dA \pm \iint_R g,dA$
区域可加性若 $R = R_1 \cup R_2$ 且 $R_1 \cap R_2$ 仅沿边界，则 $\iint_R f,dA = \iint_{R_1} f,dA + \iint_{R_2} f,dA$
比较性质若 $f(x,y) \geq g(x,y)$ 对所有 $(x,y) \in R$，则 $\iint_R f,dA \geq \iint_R g,dA$
特殊情形若 $m \leq f(x,y) \leq M$，则 $m \cdot A(R) \leq \iint_R f,dA \leq M \cdot A(R)$

15.1.4 例题：矩形区域上的二重积分

例 1：计算 $\iint_R (x^2 + y^2),dA$，其中 $R = [0,2] \times [1,3]$。

解：

$$\begin{aligned}
\iint_R (x^2 + y^2),dA &= \int_0^2 \int_1^3 (x^2 + y^2),dy,dx \
&= \int_0^2 \left[ x^2y + \frac{y^3}{3} \right]_{y=1}^{y=3} dx \
&= \int_0^2 \left( 3x^2 + \frac{27}{3} - x^2 - \frac{1}{3} \right) dx \
&= \int_0^2 \left( 2x^2 + 9 - \frac{1}{3} \right) dx = \int_0^2 \left( 2x^2 + \frac{26}{3} \right) dx \
&= \left[ \frac{2}{3}x^3 + \frac{26}{3}x \right]_0^2 = \frac{16}{3} + \frac{52}{3} = \frac{68}{3}
\end{aligned}$$

15.2 一般区域上的二重积分

(Double Integrals over General Regions)

15.2.1 区域分类

Type I 区域（垂直简单区域, Vertically Simple）

区域由上下两条曲线和左右两条竖直线界定：

$$D = {(x,y) \mid a \leq x \leq b,; g_1(x) \leq y \leq g_2(x)}$$

二重积分计算公式：

$$\iint_D f(x,y),dA = \int_{x=a}^{x=b} \int_{y=g_1(x)}^{y=g_2(x)} f(x,y),dy,dx$$

注意：Type I 总是**先积 $y$ 后积 $x$**。内层积分的上下限是 $x$ 的函数，外层是常数。

Type II 区域（水平简单区域, Horizontally Simple）

区域由左右两条曲线和上下两条水平线界定：

$$D = {(x,y) \mid c \leq y \leq d,; h_1(y) \leq x \leq h_2(y)}$$

二重积分计算公式：

$$\iint_D f(x,y),dA = \int_{y=c}^{y=d} \int_{x=h_1(y)}^{x=h_2(y)} f(x,y),dx,dy$$

注意：Type II 总是**先积 $x$ 后积 $y$**。内层积分的上下限是 $y$ 的函数，外层是常数。

15.2.2 改变积分次序 (Changing Order of Integration)

有时候，按某一次序积分会非常困难（甚至无法用初等函数表示），但交换次序后可能变得简单。交换次序的关键步骤：

  1. 画出积分区域：从原积分限中确定 $D$ 的区域描述
  2. 将区域重新描述为另一种 Type
  3. 写出新的积分限

经典陷阱：$\int_0^1 \int_x^1 e^{y^2},dy,dx$ – 先积 $y$ 时 $e^{y^2}$ 不可积（无初等原函数），但交换次序后 $\int_0^1 \int_0^y e^{y^2},dx,dy = \int_0^1 y e^{y^2},dy = \frac{1}{2}(e-1)$ 就很简单。

15.2.3 将区域拆分为更简单的部分

如果一个区域既是 Type I 又是 Type II，则两种积分次序都可行。如果区域不规则（既不是 Type I 也不是 Type II），可将其拆分为若干个子区域 $D_1, D_2, \ldots, D_k$：

$$\iint_D f,dA = \iint_{D_1} f,dA + \iint_{D_2} f,dA + \cdots + \iint_{D_k} f,dA$$

15.2.4 例题

例 2 (Type I)：计算 $\iint_D xy,dA$，其中 $D$ 是由 $y = x^2$ 和 $y = \sqrt{x}$ 围成的区域。

解：两曲线交点：$x^2 = \sqrt{x} \implies x=0,1$。区域可描述为：
$$D = {(x,y) \mid 0 \leq x \leq 1,; x^2 \leq y \leq \sqrt{x}}$$

$$\begin{aligned}
\iint_D xy,dA &= \int_0^1 \int_{x^2}^{\sqrt{x}} xy,dy,dx
= \int_0^1 x \cdot \left[ \frac{y^2}{2} \right]_{y=x^2}^{y=\sqrt{x}} dx \
&= \int_0^1 x \cdot \frac{(\sqrt{x})^2 - (x^2)^2}{2},dx
= \int_0^1 \frac{x(x - x^4)}{2},dx \
&= \frac{1}{2} \int_0^1 (x^2 - x^5),dx
= \frac{1}{2} \left[ \frac{x^3}{3} - \frac{x^6}{6} \right]_0^1
= \frac{1}{2} \left( \frac{1}{3} - \frac{1}{6} \right)
= \frac{1}{12}
\end{aligned}$$

15.3 用二重积分求面积

(Area by Double Integration)

区域 $D$ 的面积可以通过对常数函数 $f(x,y) = 1$ 的二重积分求得：

$$A = \iint_D 1,dA = \iint_D dA$$

在极坐标下：$A = \iint_D r,dr,d\theta$

15.4 极坐标下的二重积分

(Double Integrals in Polar Form)

15.4.1 极坐标变换

$$x = r \cos\theta, \quad y = r \sin\theta, \quad r^2 = x^2 + y^2$$

极坐标下的面积微元：

$$dA = r,dr,d\theta$$

为什么有 $r$？ 几何解释：在极坐标下，$\theta$ 固定变化 $\Delta\theta$，$r$ 固定变化 $\Delta r$，小区域的面积近似为 $r \cdot \Delta r \cdot \Delta\theta$（扇形面积之差）。

15.4.2 极坐标下的二重积分公式

$$\iint_D f(x,y),dA = \iint_D f(r\cos\theta,, r\sin\theta); r,dr,d\theta$$

15.4.3 适合使用极坐标的情况

  • 积分区域是圆形、环形、扇形或以原点为中心的曲线
  • 被积函数含有 $x^2 + y^2$、$\frac{y}{x}$ 等表达式
  • 积分限在直角坐标下非常复杂，但在极坐标下变为常数

15.4.4 极坐标下设定积分限

常见的极坐标区域：

  1. 整个圆盘 $x^2 + y^2 \leq a^2$：$0 \leq \theta \leq 2\pi,; 0 \leq r \leq a$
  2. 环形区域 $a^2 \leq x^2 + y^2 \leq b^2$：$0 \leq \theta \leq 2\pi,; a \leq r \leq b$
  3. 半圆盘（上半）：$0 \leq \theta \leq \pi,; 0 \leq r \leq a$
  4. 扇形：$\alpha \leq \theta \leq \beta,; 0 \leq r \leq f(\theta)$

15.4.5 例题

例 3：计算 $\iint_D e^{x^2+y^2},dA$，其中 $D$ 是圆盘 $x^2 + y^2 \leq 4$。

解：使用极坐标，$x^2+y^2 = r^2$，$dA = r,dr,d\theta$：
$$D: 0 \leq \theta \leq 2\pi,; 0 \leq r \leq 2$$

$$\begin{aligned}
\iint_D e^{x^2+y^2},dA &= \int_0^{2\pi} \int_0^2 e^{r^2} \cdot r,dr,d\theta \
&= \int_0^{2\pi} \left[ \frac{1}{2} e^{r^2} \right]_{r=0}^{r=2} d\theta \
&= \int_0^{2\pi} \frac{1}{2}(e^4 - 1),d\theta
= \pi(e^4 - 1)
\end{aligned}$$

关键：这个积分在直角坐标下无法直接计算（$e^{x^2+y^2}$ 无法分离变量后用初等函数表示内层积分），但在极坐标下非常简洁。

15.5 & 15.6 二重积分的应用：质量、矩、质心

(Applications: Mass, Moments, Center of Mass)

15.5.1 质量 (Mass)

若薄板占据区域 $D$，面密度为 $\delta(x,y)$（单位面积的质量），则总质量为：

$$m = \iint_D \delta(x,y),dA$$

15.5.2 一阶矩 (First Moments)

$$\begin{aligned}
M_x &= \iint_D y,\delta(x,y),dA \quad \text{(对 $x$ 轴的矩)} \[4pt]
M_y &= \iint_D x,\delta(x,y),dA \quad \text{(对 $y$ 轴的矩)}
\end{aligned}$$

15.5.3 质心/形心 (Center of Mass / Centroid)

$$\bar{x} = \frac{M_y}{m} = \frac{\iint_D x,\delta(x,y),dA}{\iint_D \delta(x,y),dA}$$

$$\bar{y} = \frac{M_x}{m} = \frac{\iint_D y,\delta(x,y),dA}{\iint_D \delta(x,y),dA}$$

若密度均匀（$\delta$ 为常数），则退化为形心 (centroid)，$\delta$ 可以约掉。

15.5.4 转动惯量 (Moments of Inertia / Second Moments)

$$\begin{aligned}
I_x &= \iint_D y^2,\delta(x,y),dA \quad &\text{(绕 $x$ 轴)} \[4pt]
I_y &= \iint_D x^2,\delta(x,y),dA \quad &\text{(绕 $y$ 轴)} \[4pt]
I_O &= \iint_D (x^2 + y^2),\delta(x,y),dA = I_x + I_y \quad &\text{(绕原点, polar moment)}
\end{aligned}$$

15.5.5 曲面积分求表面积 (Surface Area)

对于曲面 $z = f(x,y)$ 在区域 $D$ 上的部分，其表面积为：

$$A = \iint_D \sqrt{1 + \left(\frac{\partial z}{\partial x}\right)^2 + \left(\frac{\partial z}{\partial y}\right)^2}; dA$$

直觉理解：$\sqrt{1 + f_x^2 + f_y^2}$ 是倾斜面与水平面之间的面积比例因子。这是 $dS = \sqrt{1 + f_x^2 + f_y^2},dA$（曲面积微元）。

15.7 直角坐标下的三重积分

(Triple Integrals in Rectangular Coordinates)

15.7.1 定义

对于定义在三维区域 $D$ 上的函数 $f(x,y,z)$，三重积分定义为：

$$\iiint_D f(x,y,z),dV$$

其中 $dV = dx,dy,dz$ 是体积微元。

15.7.2 Fubini 定理在三维的推广

若 $D$ 是一个长方形盒子 $[a,b] \times [c,d] \times [p,q]$：

$$\iiint_D f,dV = \int_a^b \int_c^d \int_p^q f(x,y,z),dz,dy,dx$$

六种可能的积分次序：

  1. $dz,dy,dx$
  2. $dz,dx,dy$
  3. $dy,dz,dx$
  4. $dy,dx,dz$
  5. $dx,dy,dz$
  6. $dx,dz,dy$

策略：选择能使积分计算最简单的次序。通常将最复杂的变量放在最内层。

15.7.3 一般区域上的三重积分

对于一般的三维区域，积分限取决于变量之间的关系：

$$\iiint_D f(x,y,z),dV = \int_{x=a}^{x=b} \int_{y=g_1(x)}^{y=g_2(x)} \int_{z=u_1(x,y)}^{z=u_2(x,y)} f(x,y,z),dz,dy,dx$$

核心思想：将三维区域投影到 $xy$ 平面上，$x,y$ 的积分限由二维投影区域决定，$z$ 的积分限由上下曲面决定（通常是 $x$ 和 $y$ 的函数）。

15.7.4 用三重积分求体积

$$V = \iiint_D 1,dV = \iiint_D dV$$

15.7.5 例题

例 4：求由平面 $x=0, y=0, z=0$ 和 $x+y+z=1$ 围成的四面体（tetrahedron）的体积。

解：区域 $D$ 满足：
$$D = {(x,y,z) \mid 0 \leq x \leq 1,; 0 \leq y \leq 1-x,; 0 \leq z \leq 1-x-y}$$

$$V = \int_0^1 \int_0^{1-x} \int_0^{1-x-y} dz,dy,dx$$

先积 $z$：
$$\int_0^{1-x-y} dz = 1-x-y$$

再积 $y$：
$$\int_0^{1-x} (1-x-y),dy = \left[ (1-x)y - \frac{y^2}{2} \right]_0^{1-x} = (1-x)^2 - \frac{(1-x)^2}{2} = \frac{(1-x)^2}{2}$$

最后积 $x$：
$$\int_0^1 \frac{(1-x)^2}{2},dx = \frac{1}{2} \left[ -\frac{(1-x)^3}{3} \right]_0^1 = \frac{1}{2} \cdot \frac{1}{3} = \frac{1}{6}$$

验证：四面体体积 $V = \frac{1}{6} \times \text{底面积} \times \text{高} = \frac{1}{6} \times \frac{1}{2} \times 1 = \frac{1}{6}$ ✓

15.8 柱坐标下的三重积分

(Triple Integrals in Cylindrical Coordinates)

15.8.1 柱坐标定义

柱坐标是对 $(x,y)$ 作极坐标变换，保留 $z$ 不变：

$$x = r \cos\theta,\quad y = r \sin\theta,\quad z = z$$

其中 $r \geq 0$，$0 \leq \theta \leq 2\pi$（或 $-\pi \leq \theta \leq \pi$），$z \in \mathbb{R}$。

柱坐标下的体积微元：

$$dV = r,dz,dr,d\theta$$

注意：$dV$ 中必须有因子 $r$，这继承自极坐标的 $dA = r,dr,d\theta$。

15.8.2 适合使用柱坐标的情况

  • 积分区域关于 $z$ 轴具有旋转对称性（圆柱体、圆锥、抛物面等）
  • 被积函数含有 $x^2 + y^2$
  • 区域在 $xy$ 平面上的投影是圆形或扇形

15.8.3 常见积分次序

$$\iiint_D f,dV = \int_{\theta=\alpha}^{\theta=\beta} \int_{r=g_1(\theta)}^{r=g_2(\theta)} \int_{z=u_1(r,\theta)}^{z=u_2(r,\theta)} f(r\cos\theta, r\sin\theta, z); r,dz,dr,d\theta$$

也可以改变积分次序，例如**先 $r$ 后 $z$ 后 $\theta$**，根据具体问题选择。

15.8.4 例题

例 5：计算 $\iiint_D z,dV$，其中 $D$ 是由 $z = \sqrt{x^2 + y^2}$（锥面）和 $z = 2$（平面）围成的区域。

解：在柱坐标下，$x^2 + y^2 = r^2$，锥面为 $z = r$，区域满足：
$$0 \leq \theta \leq 2\pi,\quad 0 \leq r \leq 2,\quad r \leq z \leq 2$$

$$\begin{aligned}
\iiint_D z,dV &= \int_0^{2\pi} \int_0^2 \int_{z=r}^{z=2} z \cdot r,dz,dr,d\theta \
&= \int_0^{2\pi} \int_0^2 r \left[ \frac{z^2}{2} \right]_{z=r}^{z=2} dr,d\theta \
&= \int_0^{2\pi} \int_0^2 r \cdot \frac{4 - r^2}{2},dr,d\theta \
&= \int_0^{2\pi} \left[ r^2 - \frac{r^4}{8} \right]_0^2 d\theta \
&= \int_0^{2\pi} (4 - 2),d\theta = 2 \cdot 2\pi = 4\pi
\end{aligned}$$

15.9 球坐标下的三重积分

(Triple Integrals in Spherical Coordinates)

15.9.1 球坐标定义

$$\begin{aligned}
x &= \rho \sin\phi \cos\theta \
y &= \rho \sin\phi \sin\theta \
z &= \rho \cos\phi
\end{aligned}$$

其中：

  • $\rho \geq 0$：原点到点的距离（radial distance）
  • $0 \leq \phi \leq \pi$：与正 $z$ 轴的夹角（polar angle / colatitude）
  • $0 \leq \theta \leq 2\pi$：在 $xy$ 平面上的方位角（azimuthal angle）

重要恒等式：
$$\rho^2 = x^2 + y^2 + z^2$$

15.9.2 球坐标下的体积微元

$$dV = \rho^2 \sin\phi; d\rho,d\phi,d\theta$$

由来：$\rho^2 \sin\phi$ 是球坐标 Jacobian 行列式的绝对值。

15.9.3 适合使用球坐标的情况

  • 积分区域是球体、球壳、球冠、锥体等以原点为中心的几何体
  • 被积函数含有 $x^2 + y^2 + z^2 = \rho^2$
  • 区域边界由 $z = \pm\sqrt{x^2+y^2}$ 等锥面或 $x^2+y^2+z^2 = a^2$ 等球面描述

15.9.4 常见球坐标区域

  1. 整个球体 $\rho \leq a$：$0 \leq \theta \leq 2\pi,; 0 \leq \phi \leq \pi,; 0 \leq \rho \leq a$
  2. 上半球 $z \geq 0$：$0 \leq \theta \leq 2\pi,; 0 \leq \phi \leq \pi/2,; 0 \leq \rho \leq a$
  3. 球壳 $a \leq \rho \leq b$：$0 \leq \theta \leq 2\pi,; 0 \leq \phi \leq \pi,; a \leq \rho \leq b$

15.9.5 例题

例 6：计算 $\iiint_D (x^2 + y^2 + z^2),dV$，其中 $D$ 是球体 $x^2 + y^2 + z^2 \leq 4$。

解：在球坐标下，$x^2 + y^2 + z^2 = \rho^2$，$dV = \rho^2 \sin\phi,d\rho,d\phi,d\theta$：
$$D: 0 \leq \theta \leq 2\pi,; 0 \leq \phi \leq \pi,; 0 \leq \rho \leq 2$$

$$\begin{aligned}
\iiint_D (x^2 + y^2 + z^2),dV
&= \int_0^{2\pi} \int_0^\pi \int_0^2 \rho^2 \cdot \rho^2 \sin\phi,d\rho,d\phi,d\theta \
&= \int_0^{2\pi} \int_0^\pi \int_0^2 \rho^4 \sin\phi,d\rho,d\phi,d\theta \
&= \int_0^{2\pi} d\theta \cdot \int_0^\pi \sin\phi,d\phi \cdot \int_0^2 \rho^4,d\rho \
&= (2\pi) \cdot (2) \cdot \left( \frac{32}{5} \right)
= \frac{128\pi}{5}
\end{aligned}$$

技巧：当积分限是常数且被积函数可分离变量时，三重积分就是三个一元积分之积。

15.10 变量替换与 Jacobian

(Substitutions in Multiple Integrals)

15.10.1 Jacobian 行列式

考虑变量替换 $x = x(u,v)$，$y = y(u,v)$。变换的 Jacobian（雅可比行列式）定义为：

$$J(u,v) = \frac{\partial(x,y)}{\partial(u,v)} = \begin{vmatrix} \frac{\partial x}{\partial u} & \frac{\partial x}{\partial v} \[4pt] \frac{\partial y}{\partial u} & \frac{\partial y}{\partial v} \end{vmatrix} = \frac{\partial x}{\partial u} \cdot \frac{\partial y}{\partial v} - \frac{\partial x}{\partial v} \cdot \frac{\partial y}{\partial u}$$

15.10.2 二重积分的变量替换公式

$$\iint_D f(x,y),dx,dy = \iint_{D’} f(x(u,v),, y(u,v)); |J(u,v)|; du,dv$$

其中 $D’$ 是 $D$ 在 $(u,v)$ 坐标下的对应区域。

15.10.3 三重积分的 Jacobian

对于三维变换 $x = x(u,v,w)$，$y = y(u,v,w)$，$z = z(u,v,w)$：

$$J(u,v,w) = \frac{\partial(x,y,z)}{\partial(u,v,w)} = \begin{vmatrix} x_u & x_v & x_w \ y_u & y_v & y_w \ z_u & z_v & z_w \end{vmatrix}$$

$$\iiint_D f,dx,dy,dz = \iiint_{D’} f(x(u,v,w),\ldots); |J(u,v,w)|; du,dv,dw$$

15.10.4 常见坐标变换的 Jacobian

变换$J$
极坐标 $(r,\theta)$$J = r$
柱坐标 $(r,\theta,z)$$J = r$
球坐标 $(\rho,\phi,\theta)$$J = \rho^2 \sin\phi$

这些就是之前各节中 $dA$ 或 $dV$ 微元中的”额外因子”。

15.10.5 例题：线性变换

例 7：计算 $\iint_D (x+y),dA$，其中 $D$ 是由 $x+y=0$、$x+y=2$、$x-y=0$、$x-y=1$ 围成的平行四边形。

解：令 $u = x+y$，$v = x-y$，则：
$$D’: 0 \leq u \leq 2,; 0 \leq v \leq 1$$

求 Jacobian：由 $u = x+y$，$v = x-y$ 解得 $x = \frac{u+v}{2}$，$y = \frac{u-v}{2}$

$$J(u,v) = \begin{vmatrix} \frac{\partial x}{\partial u} & \frac{\partial x}{\partial v} \[4pt] \frac{\partial y}{\partial u} & \frac{\partial y}{\partial v} \end{vmatrix} = \begin{vmatrix} \frac{1}{2} & \frac{1}{2} \[4pt] \frac{1}{2} & -\frac{1}{2} \end{vmatrix} = -\frac{1}{4} - \frac{1}{4} = -\frac{1}{2}$$

$$|J| = \frac{1}{2}$$

$$\iint_D (x+y),dA = \int_0^1 \int_0^2 u \cdot \frac{1}{2},du,dv = \frac{1}{2} \cdot 2 \cdot 1 = 1$$

常见题型与解题策略总结

(Common Problem Types and Solution Strategies)

类型 1：计算矩形区域上的二重积分

  • 直接用 Fubini 定理，任选 $dx,dy$ 或 $dy,dx$ 次序
  • 注意积分次序的选择不影响结果

类型 2：一般区域的二重积分——确定积分限

步骤：

  1. 画出区域 $D$ 的草图
  2. 确定区域类型（Type I 还是 Type II）
  3. 写出积分限（外层常数，内层函数）
  4. 计算累次积分

类型 3：交换积分次序

步骤：

  1. 从积分限画出区域
  2. 将区域重新描述为另一种 Type
  3. 写出新的积分限

类型 4：极坐标二重积分

特征信号：

  • 被积函数含 $x^2 + y^2$
  • 区域是圆、环、扇形
  • 直角坐标下积分限复杂

步骤：

  1. 将 $x,y$ 替换为 $r\cos\theta, r\sin\theta$
  2. $dA$ 换为 $r,dr,d\theta$
  3. 确定 $r$ 和 $\theta$ 的积分限
  4. 不要忘记 $r$！

类型 5：三重积分——选择坐标系

特征推荐坐标系
长方体或一般区域，无特殊对称性直角坐标 (Rectangular)
绕 $z$ 轴旋转对称，含 $x^2+y^2$柱坐标 (Cylindrical)
球对称，含 $x^2+y^2+z^2$球坐标 (Spherical)

类型 6：计算质量、质心、转动惯量

  • 先写出密度函数 $\delta$
  • 用适当的积分公式（直角/极/柱/球）
  • 利用对称性简化计算！

类型 7：变量替换

  • 找到合适的坐标变换使积分区域变成矩形
  • 计算 Jacobian 并取绝对值
  • 代入变换后的积分公式

十大常见错误 (Top 10 Common Mistakes)

  1. **忘记极坐标/柱坐标中的 $r$**：$dA = r,dr,d\theta$，不是 $dr,d\theta$！
  2. **忘记球坐标中的 $\rho^2\sin\phi$**：$dV = \rho^2\sin\phi,d\rho,d\phi,d\theta$，不是 $d\rho,d\phi,d\theta$！
  3. 积分限写错：外层积分限必须是常数，内层才是函数。
  4. Type I/Type II 混淆：Type I 先 $dy$ 后 $dx$，Type II 先 $dx$ 后 $dy$。
  5. Jacobian 忘记取绝对值：$|J|$ 永远是非负的。
  6. 变量替换后区域映射错误：画出新旧坐标系区域的对应关系。
  7. 球坐标中 $\phi$ 的范围：$0 \leq \phi \leq \pi$（从北极到南极），不是 $0$ 到 $2\pi$。
  8. 交换积分次序时区域画错：必须准确画出原始区域。
  9. 三重积分投影到错误平面：选择使投影区域最简单的平面。
  10. 忽略对称性简化：偶函数/奇函数在对称区域上的积分可以利用对称性大大简化。
