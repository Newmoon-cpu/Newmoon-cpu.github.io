14.1 多元函数

14.1.1 二元函数的定义

一个 二元函数 (function of two variables) 是一个规则，它将有序实数对 $(x, y)$ 在集合 $D \subseteq \mathbb{R}^2$ 中的每一个元素，映射到唯一的实数 $z = f(x, y)$。

定义： 设 $D \subset \mathbb{R}^2$。在 $D$ 上的一个实值函数 $f$ 是一个规则，它为 $D$ 中每个 $(x, y)$ 指派唯一的实数 $w = f(x, y)$。集合 $D$ 为 $f$ 的 **定义域 (domain)**。

类似地，三元函数 $w = f(x, y, z)$ 将 $\mathbb{R}^3$ 中的点映射到实数。

14.1.2 定义域与值域 (Domain and Range)

  • 定义域 (Domain)： 使得函数”有意义”的所有 $(x, y)$ 的集合（通常排除除以零、平方根下负值、对数中非正数等情况）。

常见定义域类型：

函数（Function）定义域（Domain）
$f(x,y) = \sqrt{x^2 + y^2 - 1}$${(x,y) \mid x^2 + y^2 \ge 1}$ （单位圆外）
$f(x,y) = \ln(xy)$${(x,y) \mid xy > 0}$ （第一和第三象限）
$f(x,y) = \frac{1}{x^2 + y^2 - 4}$${(x,y) \mid x^2 + y^2 \ne 4}$ （去掉圆周）
  • 值域 (Range)： 函数取值的所有可能的 $z$ 值的集合。

14.1.3 函数的图形 (Graphs of $f(x,y)$)

二元函数 $z = f(x, y)$ 的图形是三维空间中的一张曲面：

$$\text{Graph of } f = {(x, y, z) \in \mathbb{R}^3 \mid z = f(x, y), (x, y) \in D}$$

常见曲面：

函数图形描述
$z = ax + by + c$平面 (Plane)一次函数
$z = x^2 + y^2$旋转抛物面 (Circular Paraboloid)向上开口的碗
$z = \sqrt{x^2 + y^2}$锥面 (Cone)顶点在原点
$z = \sqrt{r^2 - x^2 - y^2}$上半球面 (Upper Hemisphere)半径为 $r$
$z = x^2 - y^2$双曲抛物面 (Hyperbolic Paraboloid)马鞍形

14.1.4 等高线 (Level Curves)

Level curve (等高线) 是平面 $z = k$ 与曲面 $z = f(x, y)$ 的交线在 $xy$ 平面上的投影。它是方程 $f(x, y) = k$ 所定义的曲线。

  • 每条等高线对应一个常数输出值 $k$。
  • 等高线密集意味着曲面陡峭；稀疏意味着平缓。

常见等高线：

函数Level Curves $f(x,y) = k$
$z = x^2 + y^2$同心圆 $x^2 + y^2 = k$ ($k \ge 0$)
$z = x^2 - y^2$双曲线 $x^2 - y^2 = k$
$z = \frac{y}{x^2}$抛物线族

等高线图 (Contour map) 是多个等高线在同一坐标系中的图像。

14.1.5 等值面 (Level Surfaces)

对于三元函数 $w = f(x, y, z)$，等值面 (level surface) 是方程 $f(x, y, z) = k$ 所定义的曲面。

例子： $f(x, y, z) = x^2 + y^2 + z^2$ 的等值面是同心球面 $x^2 + y^2 + z^2 = k$（$k > 0$）。

14.1.6 开集与闭集 (Open and Closed Sets)

  • 圆盘 (disk)： ${(x,y) \mid (x-a)^2 + (y-b)^2 < r^2}$ — 开 (open)
  • 内点 (interior point)： 存在以该点为中心的完全包含在区域内的开圆盘
  • 边界点 (boundary point)： 每个以该点为中心的圆盘既包含区域内的点也包含区域外的点
  • 开集 (open set)： 只包含内点的集合
  • 闭集 (closed set)： 包含其所有边界点的集合
  • 有界区域 (bounded region)： 能够被某个有限圆盘包含的区域

14.2 高维空间中的极限与连续性

14.2.1 二元函数的极限

定义： 我们说 $f(x, y)$ 在 $(x, y)$ 趋近于 $(x_0, y_0)$ 时的极限为 $L$，记为
$$\lim_{(x,y) \to (x_0,y_0)} f(x,y) = L$$
如果对于任意 $\varepsilon > 0$，存在 $\delta > 0$，使得只要 $0 < \sqrt{(x-x_0)^2 + (y-y_0)^2} < \delta$，就有 $|f(x,y) - L| < \varepsilon$。

关键区别： 在一元微积分中，$(x)$ 只能从左右两个方向趋近 $x_0$。在二元微积分中，$(x, y)$ 可以从无穷多个方向和路径趋近 $(x_0, y_0)$。只有当沿着所有可能的路径极限都相同，极限才存在。

14.2.2 证明极限存在的技巧

  1. 代入法： 如果 $f$ 在 $(x_0, y_0)$ 连续，直接代入。
  2. 代数化简： 因式分解、有理化等。
  3. 夹逼定理 (Squeeze Theorem)： 如果 $|f(x,y) - L| \le g(x,y)$ 且 $\lim g(x,y) = 0$，则 $\lim f(x,y) = L$。
  4. 极坐标法： 令 $x = x_0 + r\cos\theta$, $y = y_0 + r\sin\theta$，则 $(x,y) \to (x_0,y_0)$ 等价于 $r \to 0$。如果极限存在，结果应与 $\theta$ 无关。

极坐标法示例：

求 $\displaystyle\lim_{(x,y)\to(0,0)} \frac{x^2y}{x^2+y^2}$：
令 $x = r\cos\theta$, $y = r\sin\theta$，

$$
\frac{x^2y}{x^2+y^2} = \frac{r^3\cos^2\theta\sin\theta}{r^2} = r\cos^2\theta\sin\theta \to 0 \quad (\text{as } r \to 0)
$$

因为 $|\cos^2\theta\sin\theta| \le 1$，所以极限为 $0$。

14.2.3 两路径检验法 (Two-Path Test for Non-Existence)

定理（Two-Path Test）： 如果函数 $f(x, y)$ 沿着两条不同的路径趋近 $(x_0, y_0)$ 时有不同的极限，则 $\displaystyle\lim_{(x,y)\to(x_0,y_0)} f(x,y)$ **不存在 (does not exist, DNE)**。

常用测试路径：

  1. 沿 $x$ 轴：$y = 0$
  2. 沿 $y$ 轴：$x = 0$
  3. 沿直线 $y = mx$（不同斜率）
  4. 沿抛物线 $y = kx^2$, $x = ky^2$

经典例题： 证明 $\displaystyle\lim_{(x,y)\to(0,0)} \frac{xy}{x^2+y^2}$ 不存在。

  • 路径1：$y = 0 \implies \frac{x \cdot 0}{x^2} = 0$，极限 $= 0$
  • 路径2：$y = x \implies \frac{x \cdot x}{2x^2} = \frac{1}{2}$，极限 $= \frac{1}{2}$

两个路径得到不同的极限，所以极限 DNE。

另一个经典例题： $\displaystyle\lim_{(x,y)\to(0,0)} \frac{2x^2y}{x^4+y^2}$

  • 路径 $y = mx$：极限 $= 0$（全都相同）
  • 路径 $y = x^2$：$\frac{2x^4}{x^4+x^4} = 1$，极限 $= 1$

沿不同抛物线路径得到不同极限，所以 DNE。

14.2.4 连续性

定义： 函数 $f(x, y)$ 在点 $(x_0, y_0)$ 处 **连续 (continuous)**，如果

  1. $f(x_0, y_0)$ 有定义，
  2. $\displaystyle\lim_{(x,y)\to(x_0,y_0)} f(x,y)$ 存在，
  3. $\displaystyle\lim_{(x,y)\to(x_0,y_0)} f(x,y) = f(x_0, y_0)$。

连续函数的性质：

  • 多项式函数在 $\mathbb{R}^2$ 上处处连续。
  • 有理函数在其定义域内处处连续。
  • 连续函数的和、差、积、商（分母非零）、复合仍连续。

极值定理 (Extreme Value Theorem for Two Variables)： 如果 $f$ 在有界闭区域 $R$ 上连续，则 $f$ 在 $R$ 上达到绝对最大值和绝对最小值。

14.3 偏导数

14.3.1 $\partial f / \partial x$ 和 $\partial f / \partial y$ 的极限定义

定义（关于 $x$ 的偏导数）：
$$f_x(x_0, y_0) = \frac{\partial f}{\partial x}\bigg|{(x_0,y_0)} = \lim{h \to 0} \frac{f(x_0 + h, y_0) - f(x_0, y_0)}{h}$$

定义（关于 $y$ 的偏导数）：
$$f_y(x_0, y_0) = \frac{\partial f}{\partial y}\bigg|{(x_0,y_0)} = \lim{h \to 0} \frac{f(x_0, y_0 + h) - f(x_0, y_0)}{h}$$

14.3.2 记号 (Notation)

对于一个函数 $z = f(x, y)$，其偏导数有以下几种记法：

$$f_x = \frac{\partial f}{\partial x} = \frac{\partial z}{\partial x} = D_x f$$

$$f_y = \frac{\partial f}{\partial y} = \frac{\partial z}{\partial y} = D_y f$$

14.3.3 求偏导数的规则

求 $f_x$ 时： 把 $y$ 视为常数，对 $x$ 求导。
求 $f_y$ 时： 把 $x$ 视为常数，对 $y$ 求导。

所有一元微积分的求导法则（幂法则、积法则、商法则、链式法则）都适用。

示例： 求 $f(x, y) = x^3 + x^2y^3 - 2y^2$ 的偏导数。

$$
f_x = 3x^2 + 2xy^3 \quad (\text{treat } y \text{ as constant})
$$
$$
f_y = 3x^2y^2 - 4y \quad (\text{treat } x \text{ as constant})
$$

示例： $f(x, y) = \sin\left(\frac{x}{1+y}\right)$

$$
f_x = \cos\left(\frac{x}{1+y}\right) \cdot \frac{1}{1+y}
$$
$$
f_y = \cos\left(\frac{x}{1+y}\right) \cdot \left(-\frac{x}{(1+y)^2}\right)
$$

14.3.4 偏导数的几何意义

  • $f_x(x_0, y_0)$： 曲面 $z = f(x, y)$ 在平面 $y = y_0$ 上的截线在点 $(x_0, y_0, f(x_0, y_0))$ 处的切线斜率。
  • $f_y(x_0, y_0)$： 曲面 $z = f(x, y)$ 在平面 $x = x_0$ 上的截线在点 $(x_0, y_0, f(x_0, y_0))$ 处的切线斜率。

14.3.5 高阶偏导数 (Higher-Order Partial Derivatives)

二阶偏导数有四种：

$$
\begin{aligned}
(f_x)x = f{xx} = \frac{\partial}{\partial x}\left(\frac{\partial f}{\partial x}\right) = \frac{\partial^2 f}{\partial x^2} \[4pt]
(f_y)y = f{yy} = \frac{\partial}{\partial y}\left(\frac{\partial f}{\partial y}\right) = \frac{\partial^2 f}{\partial y^2} \[4pt]
(f_x)y = f{xy} = \frac{\partial}{\partial y}\left(\frac{\partial f}{\partial x}\right) = \frac{\partial^2 f}{\partial y \partial x} \quad \text{(先对x求导，再对y求导)} \[4pt]
(f_y)x = f{yx} = \frac{\partial}{\partial x}\left(\frac{\partial f}{\partial y}\right) = \frac{\partial^2 f}{\partial x \partial y} \quad \text{(先对y求导，再对x求导)}
\end{aligned}
$$

14.3.6 克莱罗定理 (Clairaut’s Theorem)

定理 (Clairaut’s Theorem / 混合偏导数相等定理)： 如果 $f$ 定义在以 $(a, b)$ 为圆心的一个开圆盘 $D$ 上，且函数 $f_{xy}$ 和 $f_{yx}$ 在 $D$ 上连续，则
$$f_{xy}(a, b) = f_{yx}(a, b)$$

解释： 在相当温和的条件下（二阶混合偏导数连续），混合偏导数的求导次序可以交换。

更高阶的情况： 如果所有涉及的高阶偏导数都连续，例如 $f_{xxy} = f_{xyx} = f_{yxx}$。

注意： Clairaut 定理假设混合偏导数连续。如果混合偏导数不连续，$f_{xy}$ 和 $f_{yx}$ 可能不相等。

14.4 链式法则

14.4.1 情况一：$z = f(x, y)$，其中 $x = g(t)$，$y = h(t)$

如果 $z = f(x, y)$ 是关于 $x$ 和 $y$ 的可微函数，而 $x = g(t)$ 和 $y = h(t)$ 又是关于 $t$ 的可微函数，则 $z$ 是 $t$ 的函数，且：

$$
\boxed{\frac{dz}{dt} = \frac{\partial f}{\partial x}\frac{dx}{dt} + \frac{\partial f}{\partial y}\frac{dy}{dt}}
$$

树形图记忆法： $z$ 依赖 $x$ 和 $y$，$x$ 和 $y$ 各依赖 $t$。把每条路径上的导数相乘，再求和。

示例： 设 $z = x^2y + 3xy^4$，其中 $x = \sin 2t$，$y = \cos t$。求 $\frac{dz}{dt}$ 在 $t = 0$ 处的值。

先求偏导数：
$$
\frac{\partial z}{\partial x} = 2xy + 3y^4, \quad \frac{\partial z}{\partial y} = x^2 + 12xy^3
$$
$$
\frac{dx}{dt} = 2\cos 2t, \quad \frac{dy}{dt} = -\sin t
$$

然后代入 $t = 0$：
$x(0) = 0$, $y(0) = 1$
$\frac{\partial z}{\partial x}\big|{(0,1)} = 3$, $\frac{\partial z}{\partial y}\big|{(0,1)} = 0$
$\frac{dx}{dt}\big|_0 = 2$, $\frac{dy}{dt}\big|_0 = 0$

$$
\frac{dz}{dt}\bigg|_0 = 3 \cdot 2 + 0 \cdot 0 = 6
$$

14.4.2 情况二：$z = f(u, v)$，其中 $u = g(x, y)$，$v = h(x, y)$

如果 $z = f(u, v)$，且 $u$ 和 $v$ 都依赖于 $x$ 和 $y$，则 $z$ 是关于 $x$ 和 $y$ 的函数：

$$
\boxed{\frac{\partial z}{\partial x} = \frac{\partial f}{\partial u}\frac{\partial u}{\partial x} + \frac{\partial f}{\partial v}\frac{\partial v}{\partial x}}
$$

$$
\boxed{\frac{\partial z}{\partial y} = \frac{\partial f}{\partial u}\frac{\partial u}{\partial y} + \frac{\partial f}{\partial v}\frac{\partial v}{\partial y}}
$$

14.4.3 一般形式的链式法则 (General Chain Rule)

对于 $z = f(x_1, x_2, \ldots, x_n)$，其中每个 $x_i$ 又是 $m$ 个变量 $t_1, t_2, \ldots, t_m$ 的函数：

$$
\frac{\partial z}{\partial t_j} = \sum_{i=1}^{n} \frac{\partial f}{\partial x_i}\frac{\partial x_i}{\partial t_j}
$$

14.4.4 隐函数求导 (Implicit Differentiation)

情形一： 如果方程 $F(x, y) = 0$ 将 $y$ 隐式定义为 $x$ 的可微函数，且 $\frac{\partial F}{\partial y} \ne 0$，则：

$$
\boxed{\frac{dy}{dx} = -\frac{F_x}{F_y}}
$$

情形二： 如果方程 $F(x, y, z) = 0$ 将 $z$ 隐式定义为 $x$ 和 $y$ 的可微函数，且 $\frac{\partial F}{\partial z} \ne 0$，则：

$$
\boxed{\frac{\partial z}{\partial x} = -\frac{F_x}{F_z}, \quad \frac{\partial z}{\partial y} = -\frac{F_y}{F_z}}
$$

示例： 求 $y^3 + x^2y - 2x = 0$ 中 $\frac{dy}{dx}$。

令 $F(x, y) = y^3 + x^2y - 2x$
$$
F_x = 2xy - 2, \quad F_y = 3y^2 + x^2
$$
$$
\frac{dy}{dx} = -\frac{F_x}{F_y} = -\frac{2xy - 2}{3y^2 + x^2} = \frac{2 - 2xy}{3y^2 + x^2}
$$

14.5 方向导数与梯度向量

14.5.1 方向导数的定义

定义： 函数 $f$ 在点 $(x_0, y_0)$ 处沿单位向量 $\mathbf{u} = \langle a, b \rangle$（其中 $a^2 + b^2 = 1$）的方向导数为：
$$D_{\mathbf{u}} f(x_0, y_0) = \lim_{h \to 0} \frac{f(x_0 + ha, y_0 + hb) - f(x_0, y_0)}{h}$$

前提是此极限存在。

14.5.2 梯度向量 (The Gradient Vector)

定义： 如果 $f$ 是 $x$ 和 $y$ 的函数，则 $f$ 的 梯度 (gradient) 是向量函数：
$$\nabla f(x, y) = \langle f_x(x, y), f_y(x, y) \rangle = \frac{\partial f}{\partial x}\mathbf{i} + \frac{\partial f}{\partial y}\mathbf{j}$$

对于三元函数 $f(x, y, z)$：
$$\nabla f(x, y, z) = \langle f_x, f_y, f_z \rangle$$

14.5.3 方向导数与梯度的关系

定理： 如果 $f$ 是 $x$ 和 $y$ 的可微函数，则 $f$ 在点 $(x_0, y_0)$ 处沿任意单位向量 $\mathbf{u}$ 的方向导数存在，且：
$$\boxed{D_{\mathbf{u}} f(x_0, y_0) = \nabla f(x_0, y_0) \cdot \mathbf{u}}$$

将其写为点积形式，若 $\theta$ 为 $\nabla f$ 与 $\mathbf{u}$ 之间的夹角：
$$D_{\mathbf{u}} f = |\nabla f| \cdot |\mathbf{u}| \cdot \cos\theta = |\nabla f| \cos\theta$$

（因为 $|\mathbf{u}| = 1$）

14.5.4 方向导数的性质

$\nabla f$ 指向函数增加最快的方向：

当 $\theta = 0$（$\mathbf{u}$ 与 $\nabla f$ 同方向）时，$\cos\theta = 1$，$D_{\mathbf{u}} f = |\nabla f|$，达到最大值。即：

  • $\nabla f$ 指向 $f$ 增加最快的方向 (direction of maximum increase)。
  • 最大增长率 (maximum rate of change) 等于 $|\nabla f|$。

当 $\theta = \pi$（$\mathbf{u}$ 与 $\nabla f$ 反方向）时，$\cos\theta = -1$，$D_{\mathbf{u}} f = -|\nabla f|$，达到最小值（减小最快）。

当 $\theta = \pi/2$ 时，$\cos\theta = 0$，$D_{\mathbf{u}} f = 0$，函数沿此方向不变。这对应等高线的切线方向。

14.5.5 梯度与等高线的关系

定理： 梯度 $\nabla f(x_0, y_0)$ 垂直于过点 $(x_0, y_0)$ 的等高线（level curve）$f(x, y) = k$（当 $\nabla f \ne \mathbf{0}$ 时）。

解释： 在等高线上移动时，函数值不变，方向导数为零。所以梯度必须与等高线切向量正交。

14.5.6 梯度向量的代数性质

设 $f$ 和 $g$ 可微，$c$ 为常数：

$$\nabla(f + g) = \nabla f + \nabla g$$
$$\nabla(cf) = c\nabla f$$
$$\nabla(fg) = f\nabla g + g\nabla f$$
$$\nabla\left(\frac{f}{g}\right) = \frac{g\nabla f - f\nabla g}{g^2} \quad (g \ne 0)$$

14.5.7 示例

例： 求 $f(x, y) = x^2y$ 在点 $(1, 2)$ 沿方向 $\mathbf{v} = \langle 3, 4 \rangle$ 的方向导数。

第一步：求梯度 $\nabla f = \langle 2xy, x^2 \rangle$，在 $(1, 2)$ 处：$\nabla f(1,2) = \langle 4, 1 \rangle$。

第二步：将 $\mathbf{v}$ 单位化：
$$|\mathbf{v}| = \sqrt{9 + 16} = 5$$
$$\mathbf{u} = \left\langle \frac{3}{5}, \frac{4}{5} \right\rangle$$

第三步：求方向导数：
$$D_{\mathbf{u}} f(1,2) = \nabla f(1,2) \cdot \mathbf{u} = 4 \cdot \frac{3}{5} + 1 \cdot \frac{4}{5} = \frac{16}{5} = 3.2$$

14.6 切平面与微分

14.6.1 切平面的方程

曲面 $z = f(x, y)$ 在点 $(x_0, y_0, z_0)$（其中 $z_0 = f(x_0, y_0)$）处的切平面方程为：

$$\boxed{z - z_0 = f_x(x_0, y_0)(x - x_0) + f_y(x_0, y_0)(y - y_0)}$$

推导： 切平面实际上是曲面在该点附近的一阶（线性）近似。曲面在该点的法向量为：

$$\mathbf{n} = \langle f_x(x_0, y_0), f_y(x_0, y_0), -1 \rangle$$

或等价地说，曲面 $F(x, y, z) = f(x, y) - z = 0$ 的法向量为：

$$\mathbf{n} = \nabla F = \langle f_x, f_y, -1 \rangle$$

14.6.2 法线 (Normal Line)

曲面 $z = f(x, y)$ 在点 $(x_0, y_0, z_0)$ 处的 法线 (normal line) 是过该点且垂直于切平面的直线。

参数方程：
$$x = x_0 + f_x(x_0, y_0)t$$
$$y = y_0 + f_y(x_0, y_0)t$$
$$z = z_0 - t$$

14.6.3 线性化 (Linearization)

定义： $f$ 在 $(x_0, y_0)$ 处的 线性化 (linearization) 是函数：

$$\boxed{L(x, y) = f(x_0, y_0) + f_x(x_0, y_0)(x - x_0) + f_y(x_0, y_0)(y - y_0)}$$

线性化 $L(x, y)$ 是 $f(x, y)$ 在 $(x_0, y_0)$ 处的**标准线性近似 (standard linear approximation)**。

  • $z = L(x, y)$ 就是切平面。
  • 当 $(x, y)$ 接近 $(x_0, y_0)$ 时，$f(x, y) \approx L(x, y)$。

示例： 求 $f(x, y) = x^2 + y^2$ 在 $(1, 2)$ 处的线性化。

$f(1, 2) = 5$, $f_x = 2x \implies f_x(1,2) = 2$, $f_y = 2y \implies f_y(1,2) = 4$

$$
L(x, y) = 5 + 2(x - 1) + 4(y - 2) = 2x + 4y - 5
$$

用来近似 $f(1.1, 1.9)$：
$$f(1.1, 1.9) \approx L(1.1, 1.9) = 2(1.1) + 4(1.9) - 5 = 2.2 + 7.6 - 5 = 4.8$$
（真实值：$1.21 + 3.61 = 4.82$）

14.6.4 全微分 (Total Differential)

定义： 如果 $z = f(x, y)$ 可微，则 全微分 (total differential) 为：

$$\boxed{dz = f_x(x_0, y_0)dx + f_y(x_0, y_0)dy = \frac{\partial z}{\partial x}dx + \frac{\partial z}{\partial y}dy}$$

其中 $dx$ 和 $dy$ 是自变量 $x$ 和 $y$ 的微分（即 $dx = \Delta x$, $dy = \Delta y$）。

可微性定义： 如果 $\Delta z = f(x_0 + \Delta x, y_0 + \Delta y) - f(x_0, y_0)$ 可以表示为：

$$\Delta z = f_x\Delta x + f_y\Delta y + \varepsilon_1\Delta x + \varepsilon_2\Delta y$$

其中当 $(\Delta x, \Delta y) \to (0, 0)$ 时 $\varepsilon_1, \varepsilon_2 \to 0$，则我们称 $f$ 在 $(x_0, y_0)$ 处**可微 (differentiable)**。

14.6.5 可微性条件

定理（可微的充分条件）： 如果偏导数 $f_x$ 和 $f_y$ 在某开区域 $R$ 上存在，并在 $(x_0, y_0)$ 处连续，则 $f$ 在 $(x_0, y_0)$ 处可微。

可微性 $\implies$ 连续性： 如果 $f$ 在 $(x_0, y_0)$ 可微，则 $f$ 在 $(x_0, y_0)$ 连续。
（但偏导数存在 $\not\Rightarrow$ 可微性，可微性 $\not\Rightarrow$ 偏导数连续。）

关系总结：

$$\text{偏导数连续} \implies \text{可微} \implies \text{连续} \implies \text{极限存在}$$

偏导数存在是可微的必要不充分条件。

14.6.6 全微分用于误差估计

如果已知 $x$ 和 $y$ 的测量误差分别为 $\Delta x$ 和 $\Delta y$，则 $z = f(x, y)$ 的近似误差为：

$$\Delta z \approx dz = \frac{\partial z}{\partial x}\Delta x + \frac{\partial z}{\partial y}\Delta y$$

相对误差 (relative error)： $\displaystyle\frac{\Delta z}{z} \approx \frac{dz}{z}$

14.7 极值与鞍点

14.7.1 局部极值

定义： 函数 $f(x, y)$ 在点 $(a, b)$ 处有 局部极大值 (local maximum)，如果当 $(x, y)$ 接近 $(a, b)$ 时 $f(x, y) \le f(a, b)$。局部极小值 (local minimum) 的定义类似。

定理（Fermat 定理对二元函数）： 如果 $f$ 在 $(a, b)$ 处有局部极值，且一阶偏导数在该点存在，则
$$f_x(a, b) = 0 \quad \text{且} \quad f_y(a, b) = 0$$

14.7.2 临界点 (Critical Points)

定义： $(a, b)$ 是 $f$ 的 **临界点 (critical point)**，如果：

  1. $f_x(a, b) = 0$ 且 $f_y(a, b) = 0$，或
  2. 至少有一个偏导数在该点不存在。

临界点等价于 $\nabla f(a, b) = \mathbf{0}$ 或 $\nabla f$ 不存在。

14.7.3 二阶导数检验 (Second Derivative Test)

定理（二阶导数检验）： 设 $(a, b)$ 是 $f$ 的临界点（即 $f_x(a,b) = f_y(a,b) = 0$），且 $f$ 的二阶偏导数在以 $(a, b)$ 为中心的某个圆盘上连续。令

$$\boxed{D = D(a, b) = f_{xx}(a, b) \cdot f_{yy}(a, b) - [f_{xy}(a, b)]^2 = \begin{vmatrix} f_{xx} & f_{xy} \ f_{yx} & f_{yy} \end{vmatrix}}$$

（这实际上是 $f$ 的 Hessian 矩阵在 $(a, b)$ 处的行列式。）

  1. 如果 $D > 0$ 且 $f_{xx}(a, b) > 0$，则 $f(a, b)$ 是 **局部极小值 (local minimum)**。
  2. 如果 $D > 0$ 且 $f_{xx}(a, b) < 0$，则 $f(a, b)$ 是 **局部极大值 (local maximum)**。
  3. 如果 $D < 0$，则 $f(a, b)$ 不是局部极值，$(a, b)$ 是 **鞍点 (saddle point)**。
  4. 如果 $D = 0$，检验失效，需要其他方法判断。

记忆口诀：

$D$$f_{xx}$结论
$D > 0$$f_{xx} > 0$局部极小值 (Local Minimum)
$D > 0$$f_{xx} < 0$局部极大值 (Local Maximum)
$D < 0$—鞍点 (Saddle Point)
$D = 0$—不确定 (Inconclusive)

14.7.4 鞍点 (Saddle Point)

鞍点是这样一个临界点：在某些方向上函数取得局部极大值，在另一些方向上取得局部极小值。图形上类似马鞍。

经典鞍点函数： $f(x, y) = x^2 - y^2$ 在 $(0, 0)$。

  • 沿 $x$ 轴方向 ($y = 0$)：$f = x^2$，$(0,0)$ 是极小值
  • 沿 $y$ 轴方向 ($x = 0$)：$f = -y^2$，$(0,0)$ 是极大值

14.7.5 完整求解步骤

求 $f(x, y)$ 的局部极值和鞍点：

  1. 求一阶偏导数 $f_x$ 和 $f_y$。
  2. 解方程组 $f_x = 0$, $f_y = 0$，找到所有临界点。
  3. 求二阶偏导数 $f_{xx}$, $f_{yy}$, $f_{xy}$。
  4. 在每个临界点计算 $D = f_{xx}f_{yy} - (f_{xy})^2$。
  5. 应用二阶导数检验判断每个临界点的类型。

示例： 求 $f(x, y) = x^3 + y^3 - 3x - 12y + 20$ 的极值与鞍点。

步骤1： $f_x = 3x^2 - 3$, $f_y = 3y^2 - 12$

步骤2： 求解 $3x^2 - 3 = 0 \implies x = \pm 1$；$3y^2 - 12 = 0 \implies y = \pm 2$

临界点：$(1, 2)$, $(1, -2)$, $(-1, 2)$, $(-1, -2)$

步骤3： $f_{xx} = 6x$, $f_{yy} = 6y$, $f_{xy} = 0$

14.7.6 闭有界区域上的绝对极值

定理（极值定理 / Extreme Value Theorem）： 如果 $f$ 在有界闭集 (closed bounded set) $R$ 上连续，则 $f$ 在 $R$ 上达到绝对最大值和绝对最小值。

求闭有界区域上的绝对极值的步骤：

  1. 求区域 $R$ 内部的临界点（$f_x = f_y = 0$），并在这些点处计算 $f$ 的值。
  2. 求 $f$ 在区域边界上的极值（将边界参数化，转化为沿每条边的一元函数的极值问题）。
  3. 比较步骤 1 和步骤 2 中的所有函数值。最大者为绝对最大值，最小者为绝对最小值。

示例： 求 $f(x, y) = x^2 + y^2 - 2x$ 在闭圆盘 $x^2 + y^2 \le 4$ 上的绝对极值。

内部临界点： $f_x = 2x - 2 = 0 \implies x = 1$, $f_y = 2y = 0 \implies y = 0$。临界点 $(1, 0)$ 在内部。$f(1, 0) = -1$。

边界： $x^2 + y^2 = 4 \implies y^2 = 4 - x^2$，其中 $-2 \le x \le 2$。
在边界上：$f = x^2 + (4 - x^2) - 2x = 4 - 2x$。
在 $x = -2$ 时 $f = 8$，在 $x = 2$ 时 $f = 0$。

比较： 内部 $f = -1$，边界 $f = 0, 8$。

  • 绝对最小值 (Absolute Minimum)：$f(1, 0) = -1$
  • 绝对最大值 (Absolute Maximum)：$f(-2, 0) = 8$

14.8 拉格朗日乘数法

14.8.1 方法概述

问题： 求函数 $f(x, y)$ 在约束条件 $g(x, y) = k$（或 $g(x, y, z) = k$）下的最大值或最小值。

14.8.2 拉格朗日乘数法 (单个约束)

定理（Lagrange Multipliers / 拉格朗日乘数法）： 为求 $f(x, y)$ 在约束 $g(x, y) = k$ 下的极值候选点，求解方程组：

$$\boxed{\nabla f(x, y) = \lambda \nabla g(x, y)}$$
$$\boxed{g(x, y) = k}$$

其中 $\lambda$ 称为 **拉格朗日乘数 (Lagrange multiplier)**。

分量形式：

$$
\begin{cases}
f_x = \lambda g_x \[4pt]
f_y = \lambda g_y \[4pt]
g(x, y) = k
\end{cases}
$$

对于三个变量 $f(x, y, z)$ 受约束 $g(x, y, z) = k$，同理：

$$
\nabla f = \lambda \nabla g \quad \text{即} \quad f_x = \lambda g_x, ; f_y = \lambda g_y, ; f_z = \lambda g_z, ; g(x,y,z) = k
$$

14.8.3 几何解释

$\nabla f = \lambda \nabla g$ 意味着 $f$ 的梯度与 $g$ 的梯度平行（共线）。

几何意义： 在最优点处，$f$ 的等高线 $f(x, y) = c$ 与约束曲线 $g(x, y) = k$ 相切。这意味着在切点处，$f$ 的梯度 $\nabla f$ 垂直于约束曲线，而 $\nabla g$ 也垂直于约束曲线，所以 $\nabla f$ 和 $\nabla g$ 平行。

14.8.4 求解步骤

  1. 写出方程组：$\nabla f = \lambda \nabla g$，$g = k$。
  2. 求解 $x$, $y$ 和 $\lambda$（通常分情况讨论 $x$, $y$ 是否为零等）。
  3. 在所有候选点处计算 $f$ 的值。
  4. 最大值/最小值必然出现在候选点之中（如果有界闭约束曲线）或通过比较候选值确定。

14.8.5 示例

例1： 求 $f(x, y) = xy$ 在椭圆 $x^2 + 4y^2 = 4$ 上的极值。

令 $g(x, y) = x^2 + 4y^2$，约束为 $g = 4$。

$$
\nabla f = \langle y, x \rangle, \quad \nabla g = \langle 2x, 8y \rangle
$$

方程组：
$$
y = \lambda \cdot 2x \quad (1)
$$
$$
x = \lambda \cdot 8y \quad (2)
$$
$$
x^2 + 4y^2 = 4 \quad (3)
$$

由 (1)：若 $x \ne 0$，则 $\lambda = \frac{y}{2x}$。
由 (2)：若 $y \ne 0$，则 $\lambda = \frac{x}{8y}$。

令两式相等：$\frac{y}{2x} = \frac{x}{8y} \implies 8y^2 = 2x^2 \implies x^2 = 4y^2$。

代入 (3)：$4y^2 + 4y^2 = 4 \implies 8y^2 = 4 \implies y = \pm \frac{1}{\sqrt{2}}$。

$x^2 = 4y^2 = 2 \implies x = \pm \sqrt{2}$。

四个候选点：

  • $(\sqrt{2}, 1/\sqrt{2})$：$f = \sqrt{2} \cdot 1/\sqrt{2} = 1$
  • $(\sqrt{2}, -1/\sqrt{2})$：$f = -1$
  • $(-\sqrt{2}, 1/\sqrt{2})$：$f = -1$
  • $(-\sqrt{2}, -1/\sqrt{2})$：$f = 1$

还需要考虑 $x = 0$ 或 $y = 0$ 的情况：

  • 若 $x = 0$，由 (3) $4y^2 = 4 \implies y = \pm 1$。由 (2) $0 = 8\lambda y$，若 $y \ne 0$ 则 $\lambda = 0$，由 (1) $y = 0$ 矛盾。所以 $(0, \pm 1)$ 不满足方程组。
  • 若 $y = 0$，类似地排除。

结论： 最大值 $= 1$，最小值 $= -1$。

14.8.6 多个约束

对于 $f(x, y, z)$ 受两个约束 $g(x, y, z) = k$, $h(x, y, z) = c$ 的问题：

$$
\nabla f = \lambda \nabla g + \mu \nabla h
$$

其中 $\lambda$ 和 $\mu$ 是拉格朗日乘数。这是四个方程（三个分量方程加两个约束）解五个未知数 $(x, y, z, \lambda, \mu)$。

14.8.7 拉格朗日乘数法的典型应用问题

  1. 最大化/最小化面积、体积、距离 在约束条件下。
  2. 经济学中的效用最大化：Maximize $U(x, y)$ subject to $p_1x + p_2y = I$（预算约束）。
  3. 点到平面的最短距离：Minimize $d^2 = (x-x_0)^2 + (y-y_0)^2 + (z-z_0)^2$ subject to $ax + by + cz = d$。

14.9 二元函数的泰勒公式

14.9.1 二元函数的一阶泰勒近似

$f(x, y)$ 在 $(a, b)$ 附近的线性近似（即线性化）：

$$
f(x, y) \approx f(a, b) + f_x(a, b)(x - a) + f_y(a, b)(y - b)
$$

这其实正是我们在 14.6 中见到的 切平面 的方程。

14.9.2 二元函数的二阶泰勒公式

定理 (Taylor’s Formula for $f(x, y)$ at $(a, b)$)： 如果 $f$ 及其到 $(n+1)$ 阶的偏导数在以 $(a, b)$ 为中心的某圆盘上连续，则对于该圆盘内的 $(x, y)$，

$$f(x, y) = f(a, b) + \left[(x-a)f_x + (y-b)f_y\right]{(a,b)}$$
$$+ \frac{1}{2!}\left[(x-a)^2f{xx} + 2(x-a)(y-b)f_{xy} + (y-b)^2f_{yy}\right]{(a,b)}$$
$$+ \frac{1}{3!}\left[(x-a)^3f{xxx} + 3(x-a)^2(y-b)f_{xxy} + 3(x-a)(y-b)^2f_{xyy} + (y-b)^3f_{yyy}\right]_{(a,b)}$$
$$+ \cdots + R_n$$

其中下标 $(a, b)$ 表示所有偏导数在该点取值。

14.9.3 紧凑形式 (Compact Form)

令 $h = x - a$, $k = y - b$，并使用算子记号：

$$
\left(h\frac{\partial}{\partial x} + k\frac{\partial}{\partial y}\right) f = h f_x + k f_y
$$

$$
\left(h\frac{\partial}{\partial x} + k\frac{\partial}{\partial y}\right)^2 f = h^2 f_{xx} + 2hk f_{xy} + k^2 f_{yy}
$$

（其中二次项系数来自二项式展开 $(h\partial_x + k\partial_y)^2 = h^2\partial_x^2 + 2hk\partial_x\partial_y + k^2\partial_y^2$。）

则 Taylor 公式可写为：

$$\boxed{f(a+h, b+k) = \sum_{m=0}^{n} \frac{1}{m!}\left(h\frac{\partial}{\partial x} + k\frac{\partial}{\partial y}\right)^m f(a, b) + R_n}$$

其中 $R_n$ 是余项。

14.9.4 二阶近似 (Second-Degree Taylor Polynomial)

切掉高阶项后得到二次近似 (quadratic approximation)：

$$
Q(x, y) = f(a, b) + f_x(a,b)(x-a) + f_y(a,b)(y-b) + \frac{1}{2}f_{xx}(a,b)(x-a)^2 + f_{xy}(a,b)(x-a)(y-b) + \frac{1}{2}f_{yy}(a,b)(y-b)^2
$$

这对应于用二次曲面（椭圆抛物面、双曲抛物面等）来近似原函数。

14.9.5 在 $(0,0)$ 处的 Taylor 展开 (Maclaurin 展开)

当 $(a, b) = (0, 0)$ 时：

$$
f(x, y) = f(0, 0) + \left(xf_x + yf_y\right){(0,0)} + \frac{1}{2!}\left(x^2f{xx} + 2xyf_{xy} + y^2f_{yy}\right)_{(0,0)} + \cdots
$$

14.9.6 示例

例： 求 $f(x, y) = e^x \cos y$ 在 $(0, 0)$ 处的二阶 Taylor 多项式。

$f(0,0) = e^0\cos 0 = 1$

一阶偏导数：
$f_x = e^x\cos y \implies f_x(0,0) = 1$
$f_y = -e^x\sin y \implies f_y(0,0) = 0$

二阶偏导数：
$f_{xx} = e^x\cos y \implies f_{xx}(0,0) = 1$
$f_{xy} = -e^x\sin y \implies f_{xy}(0,0) = 0$
$f_{yy} = -e^x\cos y \implies f_{yy}(0,0) = -1$

二阶 Taylor 多项式：
$$
T_2(x, y) = 1 + 1\cdot x + 0\cdot y + \frac{1}{2}(1\cdot x^2 + 2\cdot 0\cdot xy + (-1)\cdot y^2) = 1 + x + \frac{1}{2}x^2 - \frac{1}{2}y^2
$$

14.9.7 近似的误差估计

余项 $R_n$ 衡量 Taylor 多项式与真实函数之间的误差：

$$|R_n(x, y)| = \left|f(x, y) - T_n(x, y)\right| \le \frac{M}{(n+1)!}\left(|x-a| + |y-b|\right)^{n+1}$$

其中 $M$ 是 $f$ 的所有 $(n+1)$ 阶偏导数在相关区域上的上界。

14.10 带约束变量的偏导数

14.10.1 问题描述

在物理和工程中，经常有变量之间存在约束关系的情况。例如：

  • 理想气体定律：$PV = nRT$（$P, V, T$ 不是独立的）
  • 变量 $x, y, z$ 满足约束 $F(x, y, z) = 0$

在求偏导数时，我们必须明确哪些变量保持常数。

14.10.2 记号约定

$\left(\frac{\partial z}{\partial x}\right)_y$ 表示在 $y$ 保持不变的情况下 $z$ 关于 $x$ 的偏导数。

这与 $\frac{\partial z}{\partial x}$ 的不同之处在于：

  • 在 $\frac{\partial z}{\partial x}$ 中，我们假设 $z = f(x, y)$，自然 $y$ 保持不变。
  • 但在有约束的情况下，如果 $x, y, z$ 通过约束 $F(x, y, z) = 0$ 关联，则 $\left(\frac{\partial z}{\partial x}\right)_y$ 与 $\left(\frac{\partial z}{\partial x}\right)_w$（$w$ 为另一变量）的值可能不同。

14.10.3 约束变量求导的关键公式

情况1： 变量 $x, y, z$ 由 $F(x, y, z) = 0$ 关联。假设 $F_z \ne 0$：

$$
\left(\frac{\partial z}{\partial x}\right)_y = -\frac{F_x}{F_z}
$$

情况2： 循环关系：

$$
\boxed{\left(\frac{\partial x}{\partial y}\right)_z \cdot \left(\frac{\partial y}{\partial z}\right)_x \cdot \left(\frac{\partial z}{\partial x}\right)_y = -1}
$$

这是热力学中非常有用的恒等式。

14.10.4 应用示例（热力学）

理想气体 $PV = nRT$（取 $n = 1$，$R$ 为常数，则 $PV = T$，设常数因子为 1）。

变量 $P, V, T$ 由 $F(P, V, T) = PV - T = 0$ 约束。

$$
F_P = V, \quad F_V = P, \quad F_T = -1
$$

$$
\left(\frac{\partial P}{\partial V}\right)_T = -\frac{F_V}{F_P} = -\frac{P}{V}
$$
$$
\left(\frac{\partial V}{\partial T}\right)_P = -\frac{F_T}{F_V} = -\frac{-1}{P} = \frac{1}{P}
$$
$$
\left(\frac{\partial T}{\partial P}\right)_V = -\frac{F_P}{F_T} = -\frac{V}{-1} = V
$$

验证循环关系：
$$
\left(\frac{\partial P}{\partial V}\right)_T \cdot \left(\frac{\partial V}{\partial T}\right)_P \cdot \left(\frac{\partial T}{\partial P}\right)_V = \left(-\frac{P}{V}\right) \cdot \left(\frac{1}{P}\right) \cdot V = -1
$$

14.10.5 常见”陷阱”

在其他教科书或上下文中，$\frac{\partial z}{\partial x}$ 可能被理解为 $x$ 变化时 $z$ 的变化率，但没有明确说明哪些变量保持不变。在热力学和物理化学中，始终使用下标来指明恒定变量。

常见题型与解题策略

题型 1：求定义域并画出定义域

问题： 求 $f(x, y) = \sqrt{9 - x^2 - y^2} + \ln(x)$ 的定义域。

策略：

  1. 分析每个部分的要求：
    • $\sqrt{9 - x^2 - y^2}$ 要求 $9 - x^2 - y^2 \ge 0 \implies x^2 + y^2 \le 9$（单位圆内）
    • $\ln(x)$ 要求 $x > 0$
  2. 取交集：${(x, y) \mid x^2 + y^2 \le 9, x > 0}$ — 这是右半边圆盘。
  3. 画出 $x^2 + y^2 = 9$ 的圆，在 $x$ 轴右侧（不含 $y$ 轴）的实心区域。

题型 3：证明极限不存在 (Two-Path Test)

经典问题： 判断 $\displaystyle\lim_{(x,y)\to(0,0)} \frac{x^2 - y^2}{x^2 + y^2}$ 是否存在。

策略：

  1. 路径 $y = 0$：$\frac{x^2}{x^2} = 1 \to 1$
  2. 路径 $x = 0$：$\frac{-y^2}{y^2} = -1 \to -1$
  3. 两个不同极限 $\implies$ DNE

关键提示：

  • 先试 $y = 0$ 和 $x = 0$。
  • 如果相同，再试 $y = mx$（一般直线）。
  • 如果仍相同，试 $y = kx^2$ 或 $x = ky^2$（曲线路径）。
  • 如果所有测试路径都得到相同的极限，可以尝试用极坐标法或夹逼定理来证明极限存在。
  • 如果找到一个路径给出不同的极限，或者极限依赖于路径参数，则极限不存在。

题型 4：求一阶和二阶偏导数

问题： 求 $f(x, y) = \ln(x^2 + y^2)$ 的所有一阶和二阶偏导数。

策略：
$$
f_x = \frac{2x}{x^2 + y^2}, \quad f_y = \frac{2y}{x^2 + y^2}
$$

$$
f_{xx} = \frac{2(x^2+y^2) - 2x \cdot 2x}{(x^2+y^2)^2} = \frac{2y^2 - 2x^2}{(x^2+y^2)^2}
$$

$$
f_{yy} = \frac{2x^2 - 2y^2}{(x^2+y^2)^2}
$$

$$
f_{xy} = f_{yx} = \frac{-4xy}{(x^2+y^2)^2}
$$

验证 Clairaut 定理： $f_{xy} = f_{yx}$，因为所有二阶偏导数在 $x^2 + y^2 > 0$ 上连续。

重要提示： $\ln(x^2 + y^2)$ 在 $(0, 0)$ 处的偏导数不存在。

题型 5：链式法则应用

问题： 设 $w = x^2 + y^2 + z^2$，其中 $x = r\cos\theta$, $y = r\sin\theta$, $z = r$。求 $\frac{\partial w}{\partial r}$ 和 $\frac{\partial w}{\partial \theta}$。

策略（使用树形图）：

$w$ 依赖于 $x, y, z$，每个都依赖于 $r$ 和 $\theta$。

$$
\frac{\partial w}{\partial r} = \frac{\partial w}{\partial x}\frac{\partial x}{\partial r} + \frac{\partial w}{\partial y}\frac{\partial y}{\partial r} + \frac{\partial w}{\partial z}\frac{\partial z}{\partial r}
$$
$$
= 2x\cos\theta + 2y\sin\theta + 2z\cdot 1
$$
$$
= 2(r\cos\theta)\cos\theta + 2(r\sin\theta)\sin\theta + 2r = 2r\cos^2\theta + 2r\sin^2\theta + 2r = 4r
$$

$$
\frac{\partial w}{\partial \theta} = \frac{\partial w}{\partial x}\frac{\partial x}{\partial \theta} + \frac{\partial w}{\partial y}\frac{\partial y}{\partial \theta} + \frac{\partial w}{\partial z}\frac{\partial z}{\partial \theta}
$$
$$
= 2x(-r\sin\theta) + 2y(r\cos\theta) + 2z \cdot 0
$$
$$
= -2(r\cos\theta)(r\sin\theta) + 2(r\sin\theta)(r\cos\theta) = 0
$$

题型 6：隐函数求导

问题： 求曲面 $x^3 + y^3 + z^3 + 6xyz = 1$ 上由 $z$ 作为 $x$ 和 $y$ 的函数时 $\frac{\partial z}{\partial x}$ 和 $\frac{\partial z}{\partial y}$。

策略：
令 $F(x, y, z) = x^3 + y^3 + z^3 + 6xyz - 1 = 0$。

$$
F_x = 3x^2 + 6yz, \quad F_z = 3z^2 + 6xy
$$
$$
\frac{\partial z}{\partial x} = -\frac{F_x}{F_z} = -\frac{3x^2 + 6yz}{3z^2 + 6xy} = -\frac{x^2 + 2yz}{z^2 + 2xy}
$$

类似地：
$$
\frac{\partial z}{\partial y} = -\frac{F_y}{F_z} = -\frac{3y^2 + 6xz}{3z^2 + 6xy} = -\frac{y^2 + 2xz}{z^2 + 2xy}
$$

题型 7：方向导数计算

问题： 求 $f(x, y) = xe^y$ 在点 $(2, 0)$ 沿从 $(2, 0)$ 到 $(-1, 4)$ 的方向的方向导数。

策略：

  1. 求方向向量：$\mathbf{v} = \langle -3, 4 \rangle$
  2. 单位化：$|\mathbf{v}| = 5$，所以 $\mathbf{u} = \langle -\frac{3}{5}, \frac{4}{5} \rangle$
  3. 求梯度：$\nabla f = \langle e^y, xe^y \rangle$，在 $(2,0)$ 处 $\nabla f(2,0) = \langle 1, 2 \rangle$
  4. 方向导数：$D_{\mathbf{u}}f = \nabla f \cdot \mathbf{u} = 1(-\frac{3}{5}) + 2(\frac{4}{5}) = \frac{5}{5} = 1$

变体问题 - 最大增长率：

  • 最大增长率的方向：$\nabla f(2,0) = \langle 1, 2 \rangle$ 的单位向量
  • 最大增长率的值：$|\nabla f(2,0)| = \sqrt{1 + 4} = \sqrt{5}$

题型 9：线性近似和误差估计

问题： 圆锥的底面半径和高分别测量为 $r = 10$ cm 和 $h = 25$ cm，可能的误差分别为 $\Delta r = \pm 0.1$ cm 和 $\Delta h = \pm 0.1$ cm。用微分估计体积 $V = \frac{1}{3}\pi r^2 h$ 的最大可能误差。

策略：
$$dV = \frac{\partial V}{\partial r}dr + \frac{\partial V}{\partial h}dh = \frac{2}{3}\pi rh;dr + \frac{1}{3}\pi r^2;dh$$

代入 $r = 10$, $h = 25$, $dr = 0.1$, $dh = 0.1$：
$$dV = \frac{2}{3}\pi \cdot 10 \cdot 25 \cdot 0.1 + \frac{1}{3}\pi \cdot 100 \cdot 0.1$$
$$= \frac{50\pi}{3} + \frac{10\pi}{3} = 20\pi \approx 62.83 \text{ cm}^3$$

最大相对误差：
$$V = \frac{1}{3}\pi \cdot 100 \cdot 25 = \frac{2500\pi}{3}$$
$$\text{相对误差} = \frac{20\pi}{2500\pi/3} = \frac{60}{2500} = 0.024 = 2.4%$$

题型 11：闭区域上的绝对极值

问题： 求 $f(x, y) = x^2 - 2xy + 2y$ 在矩形 $D = {(x, y) \mid 0 \le x \le 3, 0 \le y \le 2}$ 上的绝对极值。

策略：

步骤1： 内部临界点。
$$f_x = 2x - 2y = 0 \implies x = y$$
$$f_y = -2x + 2 = 0 \implies x = 1$$

所以 $(1, 1)$ 是内部临界点。$f(1, 1) = 1 - 2 + 2 = 1$。

步骤2： 边界。
边界由 4 条线段组成：

  • $L_1$（底边 $y = 0$, $0 \le x \le 3$）： $f(x, 0) = x^2$。在 $x = 0$ 处最小 $(0)$，$x = 3$ 处最大 $(9)$。
  • $L_2$（顶边 $y = 2$, $0 \le x \le 3$）： $f(x, 2) = x^2 - 4x + 4 = (x-2)^2$。在 $x = 2$ 处最小 $(0)$，$x = 0$ 处最大 $(4)$，$x = 3$ 处 $f = 1$。
  • $L_3$（左边 $x = 0$, $0 \le y \le 2$）： $f(0, y) = 2y$。在 $y = 0$ 处最小 $(0)$，$y = 2$ 处最大 $(4)$。
  • $L_4$（右边 $x = 3$, $0 \le y \le 2$）： $f(3, y) = 9 - 6y + 2y = 9 - 4y$。在 $y = 2$ 处最小 $(1)$，$y = 0$ 处最大 $(9)$。

步骤3： 汇总。

候选值：$(1,1)$: 1；边界上：0, 1, 4, 9。

  • 绝对最大值 (Absolute Maximum)： $9$ at $(3, 0)$
  • 绝对最小值 (Absolute Minimum)： $0$ at $(0, 0)$ 和 $(2, 2)$

题型 12：拉格朗日乘数法求约束极值

问题： 一个矩形盒子（无盖）由 12 m$^2$ 的纸板制成。求盒子的最大可能体积。

策略：
设底面边长为 $x$ 和 $y$，高为 $z$。

体积：$V = xyz$

纸板面积（底 + 四个侧面）：$xy + 2xz + 2yz = 12$（这是约束）

拉格朗日方程：
$$\nabla V = \lambda \nabla (xy + 2xz + 2yz)$$

$$V_x = yz = \lambda(y + 2z) \quad (1)$$
$$V_y = xz = \lambda(x + 2z) \quad (2)$$
$$V_z = xy = \lambda(2x + 2y) \quad (3)$$

约束：$xy + 2xz + 2yz = 12 \quad (4)$

由 (1) 和 (2) 的对称性（乘以 $x$ 和 $y$）：
$$xyz = \lambda(xy + 2xz) = \lambda(xy + 2yz)$$
$$\implies 2xz = 2yz \implies x = y \quad (\text{若 } z \ne 0)$$

由 (2) 和 (3)：
$$xyz = \lambda(x^2 + 2xz)$$
由 (3)：$xyz = \lambda(2x^2 + 2x^2) = 4\lambda x^2$（因为 $x = y$）

令这两个 $xyz$ 的表达式相等：
$$\lambda(x^2 + 2xz) = 4\lambda x^2$$
$$\implies x^2 + 2xz = 4x^2 \implies 2xz = 3x^2 \implies z = \frac{3x}{2}$$

代入约束 (4)，注意 $x = y$ 且 $z = \frac{3x}{2}$：
$$x^2 + 2x\left(\frac{3x}{2}\right) + 2x\left(\frac{3x}{2}\right) = 12$$
$$x^2 + 3x^2 + 3x^2 = 12 \implies 7x^2 = 12 \implies x = \sqrt{\frac{12}{7}} = \frac{2\sqrt{3}}{\sqrt{7}}$$

$$z = \frac{3}{2} \cdot \frac{2\sqrt{3}}{\sqrt{7}} = \frac{3\sqrt{3}}{\sqrt{7}}$$

最大体积：
$$V_{\max} = x^2 z = \frac{12}{7} \cdot \frac{3\sqrt{3}}{\sqrt{7}} = \frac{36\sqrt{3}}{7\sqrt{7}} = \frac{36\sqrt{21}}{49}$$

题型 13：点到曲面/平面的最短距离

问题： 求点 $(1, 2, 3)$ 到平面 $2x + 3y + z = 4$ 的最短距离。

策略（使用拉格朗日乘数法）：

最小化 $d^2 = (x - 1)^2 + (y - 2)^2 + (z - 3)^2$ 受约束 $2x + 3y + z = 4$。

$$\nabla(d^2) = \lambda \nabla(2x + 3y + z)$$

$$2(x - 1) = 2\lambda, \quad 2(y - 2) = 3\lambda, \quad 2(z - 3) = \lambda$$

$$\implies x = \lambda + 1, \quad y = \frac{3\lambda}{2} + 2, \quad z = \frac{\lambda}{2} + 3$$

代入约束：
$$2(\lambda + 1) + 3\left(\frac{3\lambda}{2} + 2\right) + \left(\frac{\lambda}{2} + 3\right) = 4$$
$$2\lambda + 2 + \frac{9\lambda}{2} + 6 + \frac{\lambda}{2} + 3 = 4$$
$$2\lambda + \frac{9\lambda}{2} + \frac{\lambda}{2} + 11 = 4$$
$$\left(2 + \frac{9}{2} + \frac{1}{2}\right)\lambda = -7$$
$$7\lambda = -7 \implies \lambda = -1$$

代入得点：$x = 0$, $y = \frac{1}{2}$, $z = \frac{5}{2}$

最短距离：
$$d = \sqrt{(0-1)^2 + (\frac{1}{2}-2)^2 + (\frac{5}{2}-3)^2} = \sqrt{1 + \frac{9}{4} + \frac{1}{4}} = \sqrt{\frac{14}{4}} = \frac{\sqrt{14}}{2}$$
